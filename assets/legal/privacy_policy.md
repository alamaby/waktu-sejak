# Privacy Policy
Last updated: June 17, 2026
This Privacy Policy explains how Waktu Sejak handles data in the app. Waktu Sejak is designed as a local-first app: your events, preferences, and supporter reward data are stored on your device unless you choose to export or share them.
This document is provided for transparency and is not legal advice.
## English
### 1. Data We Store on Your Device
Waktu Sejak stores the following data locally using device storage:
- Events you create, including event name, date and time, emoji, color, recurrence type, creation time, event type, and supporter reward count when applicable.
- App preferences, including language, theme mode, dashboard sort mode, and dashboard view mode.
- A local support count used to show the Supporter badge after successful Support Developer purchases.
- Home screen widget data on Android, generated from your local events so the widget can display event information.
This data stays on your device. Waktu Sejak does not operate a developer-controlled server for syncing, analytics, tracking, or account storage.
### 2. Internet and Network Access
Waktu Sejak does not request or use the Android `INTERNET` permission for its core features. All event storage, recurrence calculations, dashboard rendering, and data export/import happen on-device using local storage and the Android share sheet. The app cannot transmit your event data to a developer server.
Network access is used only by the operating system on your behalf when:
- You tap an external link in Settings (LinkedIn, GitHub, Blog, Upwork).
- You choose to share an exported JSON backup through another app or service.
- You complete a Support Developer purchase via Google Play Billing.
In each case the connection is initiated by the operating system or a third-party app, not by Waktu Sejak sending your event data to a remote endpoint.
### 3. Support Developer Purchases
Waktu Sejak offers optional one-time Support Developer purchases through Google Play Billing. Payments are processed by Google Play.
Waktu Sejak does not receive or store your payment card details, bank details, billing address, or Google account password. The app receives purchase status information from Google Play so it can complete the purchase, unlock the local supporter reward event, and update the local Supporter badge count.
Google Play may process purchase and payment information under Google's own policies. You can review Google's privacy practices at:
https://policies.google.com/privacy
### 4. Supporter Reward Event
After a successful Support Developer purchase, Waktu Sejak creates or updates a special local Supporter event. This event may include:
- A fixed supporter event ID.
- A supporter event title.
- A support count badge.
- The date and time when the reward event was created or updated.
The Supporter event is stored locally with your other events. It cannot be edited from the app UI, but you can delete it. Deleting the event does not reset the local support count.
### 5. Data Export and Import
Waktu Sejak includes data portability features:
- Export creates a JSON backup file from your local events and opens the system share sheet.
- Import lets you choose a JSON backup file through the system file picker.
The app does not upload your backup to a developer server. However, if you choose to share an exported backup through another app or service, that third party may process the file under its own privacy policy.
### 6. External Links
The Settings screen may include links such as Privacy Policy, Terms of Service, LinkedIn, GitHub, Blog, and Upwork. Opening these links leaves Waktu Sejak and may be handled by your browser or another app. Those third-party services are governed by their own privacy policies.
### 7. Analytics, Ads, and Tracking
Waktu Sejak does not include developer-operated analytics, advertising SDKs, or cross-app tracking.
### 8. Children
Waktu Sejak is not directed to children and does not knowingly collect personal information from children.
### 9. Your Choices
You can:
- Edit or delete normal events in the app.
- Delete the Supporter reward event.
- Export your event data to a JSON backup.
- Delete app data from your device through your operating system settings.
- Uninstall the app to remove locally stored app data, subject to your device and backup settings.
### 10. Changes to This Policy
This Privacy Policy may be updated when app features or data handling practices change. The latest version should be made available from the app's Privacy Policy link or project documentation.

## Bahasa Indonesia
### 1. Data yang Disimpan di Perangkat
Waktu Sejak menyimpan data berikut secara lokal menggunakan penyimpanan perangkat:
- Event yang Anda buat, termasuk nama event, tanggal dan waktu, emoji, warna, tipe pengulangan, waktu pembuatan, tipe event, dan jumlah dukungan jika berlaku.
- Preferensi aplikasi, termasuk bahasa, mode tema, mode pengurutan dashboard, dan mode tampilan dashboard.
- Jumlah dukungan lokal yang digunakan untuk menampilkan badge Supporter setelah pembelian Dukung Developer berhasil.
- Data widget layar utama di Android, yang dibuat dari event lokal agar widget dapat menampilkan informasi event.
Data ini tetap berada di perangkat Anda. Waktu Sejak tidak mengoperasikan server milik developer untuk sinkronisasi, analitik, pelacakan, atau penyimpanan akun.
### 2. Akses Internet dan Jaringan
Waktu Sejak tidak meminta maupun menggunakan permission `INTERNET` Android untuk fitur intinya. Seluruh penyimpanan event, perhitungan pengulangan, tampilan dashboard, dan ekspor/impor data terjadi di perangkat menggunakan penyimpanan lokal dan system share sheet Android. Aplikasi tidak dapat mengirimkan data event Anda ke server developer.
Akses jaringan hanya digunakan oleh sistem operasi atas nama Anda ketika:
- Anda mengetuk tautan eksternal di Pengaturan (LinkedIn, GitHub, Blog, Upwork).
- Anda memilih membagikan cadangan JSON hasil ekspor melalui aplikasi atau layanan lain.
- Anda menyelesaikan pembelian Dukung Developer melalui Google Play Billing.
Pada setiap kasus tersebut, koneksi dilakukan oleh sistem operasi atau aplikasi pihak ketiga, bukan oleh Waktu Sejak yang mengirimkan data event Anda ke endpoint jarak jauh.
### 3. Pembelian Dukung Developer
Waktu Sejak menyediakan pembelian opsional Dukung Developer satu kali melalui Google Play Billing. Pembayaran diproses oleh Google Play.
Waktu Sejak tidak menerima atau menyimpan detail kartu pembayaran, detail bank, alamat penagihan, atau password akun Google Anda. Aplikasi menerima informasi status pembelian dari Google Play agar dapat menyelesaikan pembelian, membuka reward event lokal Supporter, dan memperbarui jumlah badge Supporter lokal.
Google Play dapat memproses informasi pembelian dan pembayaran berdasarkan kebijakan Google. Anda dapat membaca praktik privasi Google di:
https://policies.google.com/privacy
### 4. Event Reward Supporter
Setelah pembelian Dukung Developer berhasil, Waktu Sejak membuat atau memperbarui event khusus Supporter secara lokal. Event ini dapat mencakup:
- ID event supporter tetap.
- Judul event supporter.
- Badge jumlah dukungan.
- Tanggal dan waktu saat reward event dibuat atau diperbarui.
Event Supporter disimpan secara lokal bersama event lainnya. Event ini tidak dapat diedit melalui UI aplikasi, tetapi dapat dihapus. Menghapus event ini tidak mengatur ulang jumlah dukungan lokal.
### 5. Ekspor dan Impor Data
Waktu Sejak menyediakan fitur portabilitas data:
- Ekspor membuat file cadangan JSON dari event lokal Anda dan membuka system share sheet.
- Impor memungkinkan Anda memilih file cadangan JSON melalui system file picker.
Aplikasi tidak mengunggah cadangan Anda ke server milik developer. Namun, jika Anda memilih membagikan file cadangan melalui aplikasi atau layanan lain, pihak ketiga tersebut dapat memproses file berdasarkan kebijakan privasinya sendiri.
### 6. Tautan Eksternal
Layar Pengaturan dapat berisi tautan seperti Kebijakan Privasi, Ketentuan Layanan, LinkedIn, GitHub, Blog, dan Upwork. Membuka tautan tersebut akan membawa Anda keluar dari Waktu Sejak dan dapat ditangani oleh browser atau aplikasi lain. Layanan pihak ketiga tersebut memiliki kebijakan privasi masing-masing.
### 7. Analitik, Iklan, dan Pelacakan
Waktu Sejak tidak menyertakan analitik yang dioperasikan developer, SDK iklan, atau pelacakan lintas aplikasi.
### 8. Anak-Anak
Waktu Sejak tidak ditujukan untuk anak-anak dan tidak secara sadar mengumpulkan informasi pribadi dari anak-anak.
### 9. Pilihan Anda
Anda dapat:
- Mengubah atau menghapus event normal di aplikasi.
- Menghapus event reward Supporter.
- Mengekspor data event ke file cadangan JSON.
- Menghapus data aplikasi dari perangkat melalui pengaturan sistem operasi.
- Menghapus aplikasi untuk menghapus data lokal aplikasi, tergantung pengaturan perangkat dan backup Anda.
### 10. Perubahan Kebijakan
Kebijakan Privasi ini dapat diperbarui ketika fitur aplikasi atau praktik penanganan data berubah. Versi terbaru sebaiknya tersedia melalui tautan Kebijakan Privasi di aplikasi atau dokumentasi proyek.
